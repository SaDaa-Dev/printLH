📸 사진 리사이징 및 프린트 프로그램 사용 방법

=== 🚀 프로그램 설치 및 실행 ===

1. 필요한 라이브러리 설치
   명령 프롬프트(cmd) 또는 PowerShell에서 다음 명령어 실행:
   
   pip install -r requirements.txt

2. 프로그램 실행
   - 직접 실행: python main.py
   - 실행 파일 생성: python build_exe.py

=== 📱 프로그램 사용 방법 ===

1. 프로그램 실행
   - PhotoResizer.exe 파일을 더블클릭하거나
   - 명령 프롬프트에서 python main.py 실행

2. 사진 종류 선택
   - 일반사진: 풍경, 인물 등 일반적인 사진 (A4 전체 크기로 리사이징)
   - 증명사진: 여권, 신분증용 사진 (A4 용지에 6장 배치)

3. 사진 파일 선택
   - "사진 파일 선택" 버튼 클릭
   - 지원 형식: JPG, PNG, BMP, GIF, TIFF 등

4. 미리보기 확인
   - 선택한 사진이 미리보기 영역에 표시됩니다

5. 사진 리사이징
   - "사진 리사이징" 버튼 클릭
   - 선택한 종류에 따라 자동으로 A4 크기에 맞게 조정됩니다

6. 프린트
   - "프린트" 버튼 클릭
   - 기본 이미지 뷰어에서 프린트 대화상자가 열립니다
   - 프린터 설정을 확인하고 인쇄하세요

=== 🎯 사진 종류별 리사이징 규칙 ===

📷 일반사진 모드:
- 원본 이미지의 비율을 유지하면서 A4 용지에 최대한 크게 맞춤
- 부족한 부분은 흰색 배경으로 채움
- 고해상도 출력 (300 DPI)

🆔 증명사진 모드:
- 표준 증명사진 크기 (35mm x 45mm)로 자동 크롭 및 리사이징
- A4 용지에 6장 배치 (2열 3행)
- 적절한 여백으로 깔끔한 배치

=== 🛠️ 실행 파일 만들기 ===

1. 필요한 도구 설치
   pip install pyinstaller

2. 실행 파일 생성
   python build_exe.py

3. 완성된 실행 파일
   - 위치: dist/PhotoResizer.exe
   - 다른 컴퓨터에서도 Python 설치 없이 실행 가능

=== 🔧 시스템 요구사항 ===

- 운영체제: Windows 7 이상 (주 타겟)
- Python: 3.7 이상 (개발 시에만 필요)
- 메모리: 최소 512MB
- 디스크 공간: 100MB 이상

=== 🎨 지원 이미지 형식 ===

- JPEG (.jpg, .jpeg)
- PNG (.png)
- BMP (.bmp)
- GIF (.gif)
- TIFF (.tiff)

=== 💡 사용 팁 ===

1. 고품질 출력을 위해 가능한 한 고해상도 이미지를 사용하세요
2. 증명사진의 경우 정면을 바라보는 사진이 가장 좋습니다
3. 프린트 시 용지 설정을 A4로 확인하세요
4. 컬러 프린터 사용 시 사진 품질 설정을 '고품질'로 선택하세요

=== 🚨 문제 해결 ===

Q: 프로그램이 실행되지 않아요
A: Python과 필요한 라이브러리가 설치되어 있는지 확인하세요

Q: 이미지를 불러올 수 없어요
A: 지원되는 이미지 형식인지 확인하고, 파일이 손상되지 않았는지 확인하세요

Q: 프린트가 안 돼요
A: 프린터가 연결되어 있고, 기본 프린터로 설정되어 있는지 확인하세요

Q: 실행 파일이 너무 커요
A: 정상적인 현상입니다. 모든 라이브러리가 포함되어 있기 때문입니다
